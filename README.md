# BRNumberField
BRNumberField is a subclass of NSTextField, based on BRNumberFormatter(subclass of NSNumberFormatter).

###Use
#####Code or IB
	BRNumberField * numberField = [[BRNumberField alloc] initWithFrame:NSMakeRect(192, 100, 96, 22)];
	numberField.max = 999;
